必填，解决浏览器跨域限制。见下方教程。
如何创建 Cloudflare Worker（免费，约 3 分钟）
1. 打开 workers.cloudflare.com，注册免费账号
2. 点 Create Application → Create Worker
3. 删掉默认代码，把仓库里 worker.js 的内容全部粘贴进去
4. 点 Deploy，复制显示的 https://....workers.dev 链接
5. 粘贴到上方「Worker URL」里，点保存即可
